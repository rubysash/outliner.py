# outliner.py

![example](https://github.com/rubysash/outliner.py/blob/main/v9.png?raw=true)

## Purpose

1. Save notes quickly on a variety of topics
2. Export Create perfectly formatted docx with proper h1,h2,h3,h4 structure.

I don't trust google task or other free software but still needed a task editor that would allow me to take notes and export them as docx.

The sample jsons are generated by AI based on intelligent questions as a skeleton for some docs I was working on.

Basically - Take notes in an organized manner

## Features

Export to Docx with perfect formatting and indexes

|Hotkey | Description |
| -- | -- |
|CTRL + 1 |H1|
|CTRL + 2 | H2|
|CTRL + 3 | H3|
|CTRL + 4 | H4|
|CTRL + up | Move Up|
|CTRL + down | Move Down|
|CTRL + d | Delete|
|F2 | Rename|

* Resize min/max
* JSON templates sharable
* Auto save on close and use
* Auto indexes(1.2.3)
* Search Filter
* SQLite based
